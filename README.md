# COVID-19 Hospital Treatment Data Analysis

## Project Overview

This project performs Exploratory Data Analysis (EDA) on a COVID-19 hospital treatment dataset to identify patterns in patient demographics, illness severity, hospital departments, and treatment outcomes.

The analysis helps understand factors affecting patient stay duration and healthcare resource utilization.

---

## Objectives

* Analyze patient demographics
* Study illness severity distribution
* Examine hospital department utilization
* Investigate factors affecting hospital stay duration
* Visualize healthcare trends using statistical charts
